# Gui-application
Developed a GUI-based Vending Machine Application using C++ to simulate the functionality of a real-world vending machine. The application provides an intuitive interface that allows users to browse snacks and soft drinks, select products, specify quantities, and complete purchases through an interactive payment system.

Key Features:
• Interactive graphical user interface (GUI) for product selection.
• Automatic price calculation based on the selected item and quantity.
• Real-time payment validation and balance calculation.
• Detection of insufficient payments with user-friendly error messages.
• Smooth and efficient transaction workflow to enhance the user experience.

**Technologies Used:**

* C++
* Windows Forms GUI
* Microsoft Visual Studio

This project strengthened my skills in object-oriented programming, GUI development, event-driven programming, user input validation, and application logic design while creating a practical desktop application.
